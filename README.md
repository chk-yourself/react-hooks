# React Hooks Library

A collection of modern, custom React hooks written in TypeScript.

## Hooks

- useIsFirstRender
- useClickOutside
- useLocalStorage
- useFocus
- useMediaQuery
- useForm
- useResizeObserver
- useHover
- useIntersectionObserver
- useFetch
- usePrevious
- useDebounce
- useScrollLock
- useIdle
- useClipboard
